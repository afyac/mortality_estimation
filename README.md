Welcome in the Mortality Estimation Folder ! 

This folder contains some retrospective and forecasting scenarios methods from a project funded by the United Nations Children’s Fund (UNICEF)’s East Africa Regional Office and the World Health Organisation’s Somalia country office, in collaboration with LSHTM and SIMAD University.
